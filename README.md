# Platform Buddy v2

Platform Buddy v2 is a clean rebuild of Platform Buddy as a mobile-first powerlifting training cockpit.

## Core purpose

Help one lifter decide, log, review, and prepare:

- HOME: what to do today
- LOG: record training quickly
- PLAN: see today's prescription
- DATA: understand the next adjustment
- MEET: prepare for competition

## Product boundary

This repository intentionally excludes VBT, video analysis, camera features, motion tracking, and character-heavy UI.

The Shiba Buddy may appear only as a quiet spotter: small, calm, and secondary to training decisions.

## Current status

Initial product charter and Codex agent rules are being established before app implementation.

## Reference repository

The previous project remains available at `imamuufit/total-academy` as a reference for useful training logic only.

Do not copy old VBT, video, camera, heavy character art, or Service Worker HTML rewriting patterns.
